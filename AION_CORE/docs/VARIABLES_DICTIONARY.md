# DICIONARIO DE ESTADOS: AION-CORE v4.5

## Logica de Derivacao Fisica (PI-POD)

1. Campo Magnetico Toroidal (B_tor)
   Formula: B_tor = (mu0 * Ip) / (2 * pi * R)

2. Pressao do Plasma (P)
   Formula: P = ne * kB * Te

3. Beta Normalizado (beta_n)
   Formula: beta_n = (Pressao / B^2) * (a * B / Ip)
